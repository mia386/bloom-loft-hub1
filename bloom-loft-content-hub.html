⚙ Supabase Settings

Connect to Supabase so all team members see live data. Find your credentials in your Supabase project under Settings → API.

How to set up:
1. Create a free Supabase project at supabase.com
2. Go to Export / Sync → SQL Setup tab and run the SQL
3. Go to Settings → API and copy your URL + anon key here
4. Click Test Connection, then Save
5. Use Export / Sync → Push Local Data to seed your database
